# Leap Year Tool (C + Python)

A simple terminal tool to:

- Check if one year is a leap year
- List all leap years in a given range

## C Version

### Compile

```powershell
gcc leap_year_tool.c -o leap_year_tool
```

### Run

```powershell
.\leap_year_tool
```

## Python Version

### Run

```powershell
py -3.12 leap_year_tool.py
```

## Leap Year Rule

A year is a leap year if:

- It is divisible by 400, or
- It is divisible by 4 but not divisible by 100
